2. Schematic

Per chain (shown for Chain 1, L channel; repeated for R and for Chains 2, 3):

Soft Clip (SC) sub-circuit:
  V_in_L ──[R_in (DRIVE-controlled)]──(−in) SC_1A ──(out)── V_sc_L
                                         (+in)=GND
                                         (out)◄──[D1+D2 anti-parallel: 1N4148WS]──(−in)
                                         (out)◄──[R_f fixed]──────────────────────(−in)

  DRIVE 1 knob (log-taper pot) varies R_in: min R_in → high gain → strong saturation.
  Anti-parallel diodes D1+D2: when signal exceeds Vf (~0.6 V), diodes conduct, clamping output.

Hard Clip (HC) sub-circuit:
  V_in_L ──[R_drive (DRIVE-controlled)]──(−in) HC_1A ──(out)── V_hc_L
                                           (+in)=GND
                                           (out)◄──[R_f]──(−in)
                                           (out)──[Z1+Z2 anti-parallel, BZX84-C5V1]──GND

  Zener pair clamps output to ±5.1 V hard.
  DRIVE 1 varies input gain; higher drive = harder clipping.
  Trim pot RV_HC_THRESH_1 adjusts threshold symmetry.

Wavefold (WF) sub-circuit (TL074, 2 halves per channel):
  V_in_L ──[gain stage: WF_1A]──► V_gain_L
  V_gain_L ──[fold stage 1: WF_1B]──► V_wf_L
  DRIVE 1 sets input gain (gain stage amplifies before fold stages).
  Trim pot RV_WF_GAIN_1 calibrates fold onset at DRIVE = 50%.

MODE select (CD4053, MUX_1):
  Panel switch SW_MODE_1 (3-pos) drives CD4053 A/B select pins directly.
  CD4053 routes selected mode output to Chain 1 output.
  Unity-gain buffer (spare TL072 half) after CD4053 to isolate 100 Ω series resistance.

Summing Amp (L channel, using SUM_AMP TL074 halves A+B):
  V_dst1_L ──[R_sum]──┐
  V_dst2_L ──[R_sum]──┼──(−in) SUM_AMP_L ──(out)──► to LP1 L input
  V_dst3_L ──[R_sum]──┘
  Gain = R_f_sum / R_sum = 0.5 (R_f_sum = R_sum / 2)
  Same structure for R channel using SUM_AMP halves C+D.

4. Design Notes

Phase 1 — Sonic Intent

Three independent distortion instances — one per APF group output — each with its own mode and drive level. Each instance processes the stereo (L+R) output of its APF chain equally. The three distorted group signals are summed after processing and passed to LP1.

This per-chain arrangement means each formant band can be distorted differently: Group 1 (low) wavefolded for complex sub-harmonic content, Group 2 (mid) soft-clipped for warmth, Group 3 (high) hard-clipped for bright edge — or any combination.

  • Soft Clip: Gentle, musical saturation. Warm odd harmonics. Sweet spot: 30–60%.
  • Hard Clip: Aggressive transistor-style clipping, strong odd harmonics. Sweet spot: 20–50%.
  • Wavefold: Buchla-style wavefolding. Complex harmonics increase with drive. Sweet spot: 40–75%.

Phase 2 — Transfer Functions

Soft Clip:
  y = tanh(d × x) / tanh(d)
  where x = normalized input (−1 to +1 = ±5 V full scale)
        d = exp(DRIVE_knob × k)   (log taper mapping)
  At d → 0: y ≈ x (linear)
  At d → ∞: y = sign(x) (approaches hard clip)
  Even harmonics: none (symmetrical clipping → odd harmonics only)

Hard Clip:
  y = clamp(d × x, −1, +1)
  d = 1: clip onset at ±full scale
  d = 5: clip at ±1 V input — strong rectangular waveform content
  Strong odd harmonics; abrupt transition (unlike soft clip)

Wavefold:
  y = fold(d × x)
  fold(z): if |z| ≤ 1: y = z
           if 1 < |z| ≤ 3: y = 2 − z (positive z)
           if 3 < |z| ≤ 5: y = −4 + z (positive z)  [repeating]
  d = 1: no folding
  d = 2: single fold
  d = 4–5: multiple folds, complex spectral content
  Mix of even and odd harmonics depending on DRIVE setting.

Phase 3 — Circuit Notes

  • Summing headroom: three chains summed at high drive can exceed op-amp rail. Gain = 0.5× per chain (R_f = R_sum/2) keeps worst-case combined output to ±7.5 V.
  • CD4053 series resistance: ~100 Ω at ±5 V supply. Buffer each CD4053 output with a unity-gain op-amp half before the summing stage.
  • Post-Dist feedback routing: each chain's output (before the summing amp) is tapped and routed to Block 3's FB DIST BLEND crossfade circuit for that group. Route as shielded traces to avoid inter-chain crosstalk.
  • MODE switching live: brief click possible from mechanical switching. Accepted behavior.
  • Diode/zener matching: use components from the same batch within each chain. Cross-chain matching less critical.
