Formula:
  V_modbus = AMOUNT × V_source + OFFSET

Example (envelope 0–10 V, AMOUNT=0.5, OFFSET=−2.5 V):
  Silence (V=0):   mod bus = 0 − 2.5 = −2.5 V
  Full level (V=10): mod bus = 5 − 2.5 = +2.5 V
  Result: mod bus swings −2.5 V to +2.5 V (centered around 0)

2. Schematic

Mod Bus Processor (MB_PROC: TL074CDT, SOIC-14, 4 halves):

Half A: Inverting summer (MB_AMP)
  V_source ──[R_src (AMOUNT pot)]──┬──(−in, pin 2) MB_AMP ──(out, pin 1)── V_modbus_inv
  V_offset ──[R_off]───────────────┘     (+in, pin 3) = GND
                                          (out)◄──[R_f]──(−in)
  R_f / R_src controls AMOUNT gain (0.2× – 5×)
  R_off and R_f control OFFSET range (±5 V)

Half B: Inverter (MB_INV, restores signal polarity)
  V_modbus_inv ──[R_inv]──(−in, pin 6) MB_INV ──(out, pin 7)── V_modbus
                            (+in, pin 5) = GND
                            (out)◄──[R_inv]──(−in)     (unity gain invert)

IC: TL074CDT (SOIC-14) — half A: summer, half B: inverter, halves C+D: spare/buffer.

Trim pots on Mod Bus Processor:
  RV_MB_ZERO:       zero offset null (output = 0V when source = 0V and OFFSET knob centered)
  RV_MB_AMOUNT_MAX: AMOUNT maximum gain calibration (target: 5.0× at full CW)

Per-Destination Attenuverter Circuit (×19, using 10× TL074 ICs total for all destinations):

V_mod ──[100 Ω 0603]──[BAT54S SOT-23 clamp]────────────────────────────── V_att_in
                                                                            │
Override jack tip ──[100 Ω]──[BAT54S]──[tip-switch lug]───────────────────┘
  (sleeve switching: when cable inserted, disconnects V_att_in from mod bus)

V_att_in ──► ATTENUVERTER pot (9mm bipolar, center detent)
  CW end: +V_att_in
  CCW end: −V_att_in (inverter op-amp: TL074 half; gain = −1)
  Wiper ──► CV summing node at parameter input

Each bipolar pot requires 1 TL074 half for the inverter.
19 destinations × 2 halves (buffer + inverter) = 38 halves → 10× TL074 ICs.

4. Design Notes

Phase 1 — Functional Specification

Primary Mod Source

  • MOD SOURCE SEL switch (3-position, panel): selects which envelope signal normalizes into the PRIMARY MOD SOURCE jack: L (ENV OUT L only) / max(L,R) (BAT54C diode-OR, tracks louder channel) / avg(L,R) ((ENV_L+ENV_R)/2 via resistor divider into buffer).
  • PRIMARY MOD SOURCE jack: tip-switching TS jack. Unplugged → selected ENV signal is active. Plugged → any external CV or audio overrides the envelope.

Mod Bus Processor Controls

NameRangeDefaultTaperDescription
AMOUNT0.2× – 5×1×LinearScales the mod source signal before adding offset
OFFSET±5 V0 VLinearDC offset added after scaling; shifts the mod signal up or down

Formula: V_modbus = AMOUNT × V_source + OFFSET

Per-Destination Circuit

  1. OVERRIDE JACK: tip-switching input jack. When empty, the mod bus signal feeds through. When a cable is inserted, the override jack signal replaces the mod bus for this destination.
  2. 100 Ω + BAT54 clamp: standard input protection on the selected signal.
  3. ATTENUVERTER: bipolar pot (−1× to +1×, center detent at 0). Full CCW: inverts the mod signal. Center: cuts mod to zero. Full CW: passes mod signal at full scale.
  4. CV summing node: attenuverter output sums with the parameter's panel knob voltage at the filter/distortion/APF control input.

Phase 2 — Analog Behavior Model

Mod Bus Processor:
  V_modbus = AMOUNT × V_source + OFFSET
  AMOUNT ∈ [0.2, 5.0] (pot position mapped linearly)
  OFFSET ∈ [−5 V, +5 V] (pot position mapped linearly)

Circuit: inverting summer op-amp + inverter to restore polarity:
  V_modbus_inv = −(R_f/R_src × V_source + R_f/R_off × V_offset)
  V_modbus = −V_modbus_inv  (unity inverter)

Per-Destination Attenuverter:
  V_dest_cv = ATTENUVERTER_gain × V_selected
  ATTENUVERTER_gain ∈ [−1, +1]
  Center (gain=0): V_dest_cv = 0 — mod bus is fully cut
  Full CW (gain=+1): V_dest_cv = V_selected
  Full CCW (gain=−1): V_dest_cv = −V_selected (inverted)

Phase 3 — Circuit Notes

  • 19 override jacks: all jacks must be TS type with sleeve-switching lug connected to the mod bus. Verify Thonkiconn footprint has switching lug accessible in PCB layout.
  • Op-amp count: 1× TL074 for the mod bus processor + 10× TL074 for 19 attenuverter inverter stages (38 halves, 2 spare). SOURCE switch removed — replaced by FB DIST BLEND continuous knob with resistive op-amp crossfade in Block 3.
  • 19 attenuverter pots: evaluate during panel layout whether all 19 can fit comfortably within their respective zones.
