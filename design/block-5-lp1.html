  V_ctrl_L = CUTOFF knob + CUTOFF CV (attenuverter) → THAT340 matched NPN pair
  V_ctrl_R = CUTOFF knob + CUTOFF CV + V_spread (STEREO SPREAD OFFSET)
  I_abc = I_ref × e^(V_ctrl / V_T)
  +1V on V_ctrl → I_abc × 2 → ω₀ × 2 → 1 octave up

STEREO SPREAD OFFSET (R channel only):
  Bipolar pot → V_spread → summing node on R channel expo converter input
  V_spread = 0 V: both channels identical (center detent)
  V_spread > 0 V: R cutoff shifts up relative to L
  V_spread < 0 V: R cutoff shifts down relative to L
  Trim pot RV_LP1_SPREAD_NULL nulls center detent to 0 V

RESONANCE knob → inverting Iabc driver (IRES_AMP) → LP1_Q_OTA (LM13700 cell A of IC_Q_AB) → sets Q
  IC_Q_AB: cell A = LP1 Q; cell B = LP2 Q (separate IC from integrators)
  Q = 52 mV / (Iabc × R_in): more Iabc → more BP damping → LOWER Q (flatter)
  Self-oscillation at Iabc → 0 (NOT at max Iabc)
  RESONANCE = 0: Iabc = 0.74 µA, Q ≈ 0.7 (Butterworth, flat response)
  RESONANCE = max: Iabc → 0, Q → ∞ (self-oscillation)
  Inverting driver (IRES_AMP, spare TL072 half):
    V_Iabc = V_bias − V_RES_scaled  (V_bias ≈ +0.74 V; R_Iabc = 1 MΩ)

BAND OUT tap: taken at LP1 V_LP output (before LP2 input) → to Block B BAND OUT buffers

Signal levels:
  In:  ±5 V audio (from Block VCA)
  LP out: ±5 V (±8 V at high resonance peak)
  f_ref at V_ctrl=0V: 630 Hz
  Sweep range (±5V CV): 20 Hz (−5V) to ~20 kHz (+5V)

4. Design Notes

Phase 1 — Sonic Intent

First voltage-controlled lowpass filter stage. Receives signal from Block VCA, which gates or accents the signal before it enters this filter. Post-distortion tone shaping — takes the rich harmonic content from the distortion chains and sculpts it into the desired texture. At low resonance and high cutoff: transparent. At high resonance and swept cutoff: classic filter sweep. At maximum resonance: self-oscillation produces a sine-wave tone at the cutoff frequency, tunable via 1V/oct CV.

STEREO SPREAD OFFSET: bipolar knob that skews the R channel cutoff relative to L. At center (0V): both channels identical. Positive: R cutoff rises above L, creating a brighter right channel. Negative: R cutoff falls below L. This creates a static stereo width from a single CUTOFF setting, and interacts musically with modulation on the shared CUTOFF CV.

LP1's output is also available as BAND OUT (Block B) — the signal tapped here is post-LP1 but pre-LP2, capturing the first lowpass stage's character before the second LP stage further shapes it.

Phase 2 — Transfer Function

H_LP(s) = ω₀² / (s² + (ω₀/Q) × s + ω₀²)

ω₀_L = g_m / C = (I_abc / 2V_T) / C     (L channel)
ω₀_R = g_m_R / C                          (R channel, with V_spread offset)

1V/oct: V_ctrl +1V → I_abc ×2 → ω₀ ×2 → f ×2

STEREO SPREAD OFFSET:
  ω₀_R = ω_ref × 2^((V_ctrl + V_spread) / 1V)
  V_spread = 0 V → ω₀_R = ω₀_L (center detent)
  V_spread = +1 V → R cutoff 1 octave above L
  V_spread = −1 V → R cutoff 1 octave below L

f_ref = 630 Hz at V_ctrl = 0V
Sweep: −5V → ~20 Hz; +5V → ~20 kHz  (10 octaves, 2^10 = 1024×)

Component derivation:
  At I_abc_nominal = 10 µA:
  g_m = 10µA / (2 × 26mV) = 192 µS
  C = g_m / ω₀_ref = 192µS / (2π × 630) = 48.5 nF → use 47 nF (C0G)

Q control via LM13700 OTA (IC_Q_AB, cell A per channel):
  I_out injected into SUM_AMP (−in) virtual-ground → acts as BP feedback current
  Q = 52 mV / (Iabc × R_in) = 52 mV / (Iabc × 100 kΩ)
  More Iabc → more damping → LOWER Q; Iabc → 0 → Q → ∞ (self-oscillation)
  At Iabc = 0.74 µA: Q ≈ 0.7 (Butterworth flat)
  At Iabc ≈ 10 nA:   Q ≈ 52  (near self-oscillation onset)
  Inverting Iabc driver (IRES_AMP, spare TL072 half):
    V_Iabc = V_bias − V_RES_scaled  (V_bias ≈ +0.74 V; R_Iabc = 1 MΩ)
    RESONANCE=0 → Iabc=0.74 µA (Q≈0.7); RESONANCE=max → Iabc→0 (Q→∞)
  RV_LP1_QMAX trims V_bias; sets flat-response Q and self-oscillation onset.

Phase 3 — Circuit Notes

  • THAT340 expo converter: matched NPN pair required for stable 1V/oct tracking across temperature. Calibrate at room temperature; verify at 10°C and 40°C extremes.
  • STEREO SPREAD OFFSET summing node: V_spread is summed into the R channel expo converter input. The summing resistors must match to preserve 1V/oct tracking on R. Place the summing node physically close to the R expo converter input to minimize parasitic coupling.
  • LM13700 Q Iabc sizing and direction: Q = 52 mV / (Iabc × 100 kΩ). More Iabc = lower Q (more damping, flatter). Self-oscillation at Iabc → 0 — the RESONANCE control MUST decrease Iabc as it increases. IRES_AMP (inverting TL072 summer): V_Iabc = V_bias − V_RES_scaled, R_Iabc = 1 MΩ. At RESONANCE=0: Iabc=0.74 µA (Q≈0.7). At RESONANCE=100%: Iabc→0 (Q→∞). RV_LP1_QMAX trims V_bias.
  • Self-oscillation amplitude: at Q → ∞, output amplitude limited by OTA/op-amp clipping. Optionally add BAT54 anti-parallel diodes across feedback resistor in SUM_AMP to soft-limit self-oscillation amplitude.
  • IC_Q_AB pairing: LP1 Q uses cell A; LP2 Q uses cell B of the same LM13700. Keep Iabc drive traces separate to avoid crosstalk between LP1 and LP2 Q controls.
  • BAND OUT tap: take the tap at the LP1 output buffer output (unity-gain follower on V_LP). This is the same node that feeds LP2 input — both can be driven from the same buffer without loading concerns given the 1 kΩ series output resistors in Block B.
  • LM13700 linearizing diodes: connect linearizing diode inputs to OTA differential inputs via 1 kΩ series resistors for low distortion at high signal levels.
